#  📌 TodoApp
The app allows:

- Registering and logging UI within Firebase  
- Adding new tasks  
- Marking tasks as completed 
- Deleting tasks that are no longer needed  


---

## Technologies Used
- **Swift**  
- **Xcode**  
- **Firebase**
